# Week 2 - My Money

# Description

ให้น้องทำโปรแกรมสมุดบัญชีธนาคารตัวเอง โดยสามารถถอนเงินหรือฝากเงินก็ได้เรื่อยๆ หรือจะสั่งปิดบัญชีก็ได้ โดยจะหยุดการทำงานก็ต่อเมื่อรับคำว่า Stop โดย withdraw คือ การถอนเงิน deposit คือ ฝากเงิน delete คือ การปิดบัญชี (เงินคงเหลือเป็น 0)

# Specification

Input Output
บรรทัดแรก ชื่อบัญชี
อีก n บรรทัด เป็นการทำรายการในบัญชี จนกว่าจะรับคำว่า Stop
โดยบรรทัดแรก เป็นสถานะ deposit, withdraw หรือ delete
บรรทัดต่อมา จำนวนเงินเป็นจำนวนเต็ม		 ชื่อบัญชี : จำนวนเงินที่คงเหลือในธนาคาร เป็นจำนวนเต็ม
